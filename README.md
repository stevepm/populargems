Setup Instructions
==================
1. Clone this repository
1. `bundle`
1. Create databases: `rake db:create`
1. Run migrations: `rake db:migrate`
1. Run Specs `rspec`
1. Run Local Server `rails s`

On Heroku
---------
1. `heroku run 'rake db:migrate' -a HEROKUAPP`
